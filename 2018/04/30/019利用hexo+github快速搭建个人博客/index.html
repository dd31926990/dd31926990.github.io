软件安装

安装Nodejs 下载安装Nodejs 因为我是在windows下操作的所以下载的是windows的安装版 安装的过程非常简单, 直接下一步即可

安装Git 下载安装Git Git的安装也是直接下一步即可

安装Hexo 如果你已经安装了Nodejs和Git, 接下来就可以使用命令来安装Hexo了 因为是使用命令在线安装的所以一定要有网络连接 打开CMD窗口输入：npm install -g hexo-cli
